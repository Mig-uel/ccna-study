# Jeremy's IT Lab CCNA Course

This repository contains the course materials from [Jeremy's IT Lab](https://youtube.com/playlist?list=PLxbwE86jKRgMpuZuLBivzlM8s2Dk5lXBQ&si=QyxsVUcKcsXQCaH7) CCNA course. The materials include lecture notes, lab exercises, and practice exams to help students prepare for the CCNA certification.
